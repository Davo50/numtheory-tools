🔢 Факторизация чисел

Метод Ферма: ищет представление N = x² − y², т.е. N = (x−y)(x+y). Перебирает x начиная с ⌈√N⌉ и проверяет, является ли x²−N полным квадратом.
p−1 метод Полларда: вычисляет k = LCM(1,...,B), затем для a = 2,3,... проверяет GCD(a^k − 1, N). Работает, если у p−1 (для делителя p) все простые множители ≤ B.
ρ-метод Полларда: использует произвольный многочлен f(x) mod N и алгоритм Флойда (черепаха и заяц) для поиска цикла. Находит делитель через GCD(|x_i − x_{2i}|, N).
Метод Ленстры (ECM): использует арифметику эллиптических кривых над ℤ/Nℤ. Вычисляет k·P, где k — произведение простых ≤ B. Делитель находится, когда обратный элемент не существует mod N.
Квадратичное решето (verbose): строит фактор-базу, собирает B-гладкие значения Q = v² − N, затем решает систему над GF(2) и выводит все этапы: построение базы, каждую попытку получить отношение, линейную алгебру и проверку зависимостей.

Дополнительных параметров не требуется.

Используйте x как переменную. Степень: **. Итеративная функция: f(v) = (многочлен при x=v) mod N.
Примеры: x**2 + 1 x**2 + 5 x**3 + x + 1 x**2 - 1 x**2 + 3*x + 2
Больше B и max_iterations дают выше шанс факторизации, но шагов в логе будет больше.
