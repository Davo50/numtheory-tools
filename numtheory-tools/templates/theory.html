Лекция 1. Теория делимости и базовые алгоритмы

Длина числа и сложность базовых операций

Для основания $q > 0$ любое $x \in \mathbb{N}_0$ представимо: $x = \sum_{i=0}^{n-1} x_i q^i$, $0 \leq x_i < q$, $x_{n-1} \neq 0$.
$L_p(x) = \lceil \log_p(x) \rceil$ — длина числа $x$ в $p$-ичной системе.
ОперацияСложность (битовые операции)
Сложение, вычитание$O(L_2(N))$
Умножение в столбик$O(L_2(N)^2)$
Умножение Карацубы$O(L_2(N)^{\log_2 3})$
Деление с остатком$O(L_2(N)^2)$

Алгоритм Евклида

(Ламе, 1844) Число шагов алгоритма Евклида при нахождении $\mathrm{НОД}(a, b)$ не превосходит $5 \lg \min(a, b)$.

Идея доказательства: наихудший случай — последовательность Фибоначчи $1, 2, 3, 5, 8, 13, \ldots$. Для $\mathrm{НОД}(a_{n+1}, a_{n+2})$ нужно ровно $n$ шагов. Из $a_n \geq \alpha^n$ (где $\alpha = \frac{1+\sqrt 5}{2}$) и $\alpha^5 > 10$ следует $n < 5 \log_{10} b$.

Бинарный алгоритм Евклида.
  1. Пока $t > 0$:
  2.   $t \leftarrow |a - b|$
  3.   Пока $a \bmod 2 = 0$: $a \leftarrow a/2$
  4.   Если $a \geq b$: $a \leftarrow t$, goto 1; иначе $b \leftarrow t$, goto 1.
  5. Вернуть $a$.

Расширенный алгоритм Евклида и коэффициенты Безу

Ведём тройки $(x_i, y_i, z_i)$: $$(x_{-1}, y_{-1}, z_{-1}) = (1, 0, a), \quad (x_0, y_0, z_0) = (0, 1, b),$$ $$(x_i, y_i, z_i) = (x_{i-2}, y_{i-2}, z_{i-2}) - q_i (x_{i-1}, y_{i-1}, z_{i-1}),$$ где $q_i = \lfloor z_{i-2}/z_{i-1}\rfloor$. Тогда $a x_i + b y_i = z_i$, а при $z_n = 0$: $z_{n-1} = \mathrm{НОД}(a,b)$, коэффициенты Безу $= (x_{n-1}, y_{n-1})$.

Непрерывные (цепные) дроби

Любое рациональное $\alpha = a/b$ раскладывается: $$\alpha = q_1 + \cfrac{1}{q_2 + \cfrac{1}{q_3 + \cfrac{1}{\ddots}}} = [q_1; q_2, q_3, \ldots, q_n],$$ где $q_i$ — частные в алгоритме Евклида.

Подходящие дроби $P_s/Q_s$: $$P_s = q_s P_{s-1} + P_{s-2}, \quad Q_s = q_s Q_{s-1} + Q_{s-2},$$ $(P_{-1}, Q_{-1}) = (1, 0)$, $(P_0, Q_0) = (q_1, 1)$.

Быстрое возведение в степень (бинарный метод)

Для $e = \sum e_i 2^i$: $x^e = \prod x^{e_i \cdot 2^i}$. Сложность — не более $L_2(e)$ умножений и $L_2(e)$ возведений в квадрат.

Алгоритм Карацубы

Для $n = 2k$-битных $x, y$: $x = x_1 \cdot 2^k + x_0$, $y = y_1 \cdot 2^k + y_0$.

$$xy = (2^{2k} + 2^k) x_1 y_1 + (x_1 - x_0)(y_1 - y_0) \cdot 2^k + (2^k + 1) x_0 y_0.$$ 3 умножения $k$-битных чисел вместо 4.
$T(2n) \leq 3T(n) + cn \implies T(n) = O(n^{\log_2 3}) \approx O(n^{1.585})$.

Умножение по Монтгомери

Пусть $R > N$, $\mathrm{НОД}(R, N) = 1$, $RR' - NN' = 1$, $0 < R' < N$. $$x \otimes y = xy R^{-1} \pmod N, \quad \varphi_R(x) = xR^{-1} \pmod N.$$ $\varphi_R$ — изоморфизм колец $(\mathbb{Z}_N, +, \cdot)$ и $(\mathbb{Z}_N, +, \otimes)$.
Редукция Монтгомери $\varphi_R(x)$:
  1. $m = xN' \pmod R$
  2. $t = (x + mN) / R$
  3. Если $0 \leq t < N$: вернуть $t$, иначе $t - N$.
Если $R = 2^k$, деление на $R$ — сдвиг битов.

Лекция 2. Факторизация с экспоненциальной сложностью

Задача факторизации. Для составного $N$ найти $N = p_1^{\alpha_1} \cdots p_k^{\alpha_k}$. В RSA: $N = pq$, где $p, q$ — большие простые.

Метод Ферма

Если $N = pq$ нечётное, то $x = \frac{p+q}{2}$, $y = \frac{p-q}{2}$, $N = x^2 - y^2$.

Для $y = 1, 2, \ldots$: проверить, является ли $N + y^2$ полным квадратом. Если да — $N = (\sqrt{N+y^2} - y)(\sqrt{N+y^2} + y)$.

Эффективен, когда $p$ и $q$ близки. Худший случай — $O(\sqrt N)$.

Алгоритм $p-1$ Полларда

Число $N$ — $B$-гладкое, если все его простые делители $\leq B$.
$N$ — $B$-степенно-гладкое, если все его примарные делители (степени простых) $\leq B$.
  1. $a \xleftarrow{R} \mathbb{Z}_N^*$
  2. Выбрать границу $B$
  3. $k \leftarrow \mathrm{НОК}(1, 2, \ldots, B)$
  4. $a_k \leftarrow a^k \pmod N$
  5. $d \leftarrow \mathrm{НОД}(a_k - 1, N)$
  6. Если $1 < d < N$: вернуть $[d, N/d]$, иначе повторить.

Идея: если $p | N$ и $p-1$ — $B$-гладкое, то $a^{p-1} \equiv 1 \pmod p$ (малая теорема Ферма), значит $p | a^k - 1$ для $k = \mathrm{НОК}(1, \ldots, B)$.

Сложность: $O(\sqrt N \log^c N)$.

$\rho$-алгоритм Полларда

Итерация $x_i = f(x_{i-1})$ с $f(x) = x^2 + c \pmod N$. Ищут коллизию $x_i \equiv x_j \pmod p$.

Пусть $p | N$, $p \leq \sqrt N$. Если $f$ статистически «похож» на случайное отображение, $\rho$-метод находит $p$ за $O(\sqrt[4]{N} \log^3 N)$.

Идея парадокса дней рождения: в случайной последовательности длины $O(\sqrt p)$ ожидаема коллизия mod $p$. Находят алгоритмом Флойда (черепаха и заяц): $\mathrm{НОД}(x_i - x_{2i}, N)$.

$N = 1359331$, $f(x) = x^2 + 5$, $x_0 = 1$. Через 8 итераций: $\mathrm{НОД}(x_7 - x_{14}, N) = 1181$, $N = 1181 \cdot 1151$.

Лекция 3. Кольца: факториальные и евклидовы

Кольцо $(R, +, \cdot)$: $(R, +)$ — абелева группа, выполнена дистрибутивность.
Идеал $I \trianglelefteq R$ — подкольцо, замкнутое относительно умножения на $R$.
Простой идеал: $ab \in I \Rightarrow a \in I$ или $b \in I$.
Максимальный идеал: не содержится ни в каком идеале, кроме $R$.

Факториальные кольца (UFD)

Факториальное кольцо — целостное кольцо, в котором каждый ненулевой необратимый элемент однозначно (с точностью до ассоциированности и порядка) раскладывается в произведение неприводимых.

Евклидовы кольца

Евклидово кольцо $R$ — целостное кольцо с функцией нормы $\Delta: R \setminus \{0\} \to \mathbb{N}_0$:
  1. $\Delta(ab) \geq \Delta(a)$ для $a, b \neq 0$,
  2. Для любых $a, b \neq 0$ существуют $q, r$: $a = bq + r$, $r = 0$ или $\Delta(r) < \Delta(b)$.
Всякое евклидово кольцо — факториальное. Обратное неверно.

Примеры: $\mathbb{Z}$ — евклидово ($\Delta = |\cdot|$); $K[X]$ — евклидово ($\Delta = \deg$); $\mathbb{Z}[i]$ — евклидово ($\Delta(a+bi) = a^2 + b^2$).

Классический контрпример к факториальности в $\mathbb{Z}[\sqrt{-5}]$: $$6 = 2 \cdot 3 = (1+\sqrt{-5})(1-\sqrt{-5}).$$ Все четыре множителя неприводимы (через норму $N(a+b\sqrt{-5}) = a^2 + 5b^2$). Значит $\mathbb{Z}[\sqrt{-5}]$ не факториально.

Лекция 4. Современные алгоритмы факторизации

Факторизация Ленстры (ECM, Elliptic Curve Method)

Аналог $p-1$ Полларда, но вместо $\mathbb{Z}_p^*$ используется группа точек $E(\mathbb{F}_p)$.

  1. Выбрать точку $P(x_0, y_0)$ и параметр $a$.
  2. $b = y_0^2 - x_0^3 - a x_0 \pmod N$. Кривая: $Y^2 = X^3 + aX + b$.
  3. $k = \mathrm{НОК}(1, \ldots, B)$. Вычислить $kP$ через сложение точек.
  4. Если встретится необратимый знаменатель mod $N$ — $\mathrm{НОД}(\text{знаменатель}, N)$ — нетривиальный делитель.
$N = 187$, $B = 100$, $P = (0, 5)$, $a = 1 \Rightarrow b = 25$. При вычислении $6P$: $\lambda = \frac{3 \cdot 124^2 + 1}{2 \cdot 176} = \frac{127}{165}$, $\mathrm{НОД}(165, 187) = 11$, откуда $187 = 11 \cdot 17$.

Сложность: $L_p[1/2, \sqrt 2]$. Где $L_N[\alpha, c] = \exp(c (\log N)^\alpha (\log \log N)^{1-\alpha})$.

Квадратичное решето (QS)

Идея: найти $x, y$: $x^2 \equiv y^2 \pmod N$, $x \not\equiv \pm y$. Тогда $\mathrm{НОД}(x - y, N)$ — нетривиальный делитель.

Рассматривают $Q(X) = (X + \lceil\sqrt N\rceil)^2 - N$. Значения $Q(X)$ — квадраты mod $N$. Собирают $B$-гладкие значения и через линейную алгебру $\mathbb{F}_2$ находят подмножество, произведение которых — полный квадрат.

$N = 1829$, $B = 13$. $\mathcal{FB} = \{-1, 2, 5, 7, 11, 13\}$, $\lceil\sqrt{1829}\rceil = 43$. Находим: $(94 \cdot 127)^2 \equiv (2 \cdot 5 \cdot 7 \cdot 11 \cdot 13)^2$, $\mathrm{НОД}(138 \pm 1750, 1829) = \{31, 59\}$. $1829 = 31 \cdot 59$.

Сложность: $L_N[1/2, 1]$. Лучшее применение — RSA-129 (1994). Для $N > 10^{110}$ уступает NFS.

Решето числового поля (NFS)

Преемник QS, использует алгебраическую теорию чисел:

  1. Выбирают неприводимый $f(X) \in \mathbb{Z}[X]$ степени $d$ и $m$: $f(m) \equiv 0 \pmod N$.
  2. $\alpha$ — корень $f$. Работают в $\mathbb{Z}[\alpha]$: есть гомоморфизм $\varphi: \mathbb{Z}[\alpha] \to \mathbb{Z}_N$, $\varphi(\alpha) = m$.
  3. Ищут пары $(a, b)$: $a + bm$ и $a + b\alpha$ одновременно гладкие.
  4. Через линейную алгебру находят произведения-квадраты с обеих сторон.
  5. $x^2 \equiv y^2 \pmod N \Rightarrow \mathrm{НОД}(x - y, N)$.

Сложность: $L_N[1/3, c]$. Последний рекорд: RSA-250 (829 бит).

Лекция 5. Дискретный логарифм

Задача ДЛ. В группе $G = \langle g \rangle$ порядка $n$: дано $y = g^x$, найти $x = \log_g y$.

Простой перебор: $O(n)$ операций, $O(1)$ памяти.

Алгоритм Гельфонда-Шэнкса (BSGS)

  1. $s = \lceil \sqrt n \rceil$
  2. Малый шаг: $S = \{(y g^r, r) : r = 0, \ldots, s-1\}$
  3. Большой шаг: $T = \{(g^{ts}, ts) : t = 0, \ldots, s\}$
  4. Найти $y g^r = g^{ts} \Rightarrow x = ts - r$.
Сложность: $O(\sqrt n \log n)$ операций, $O(\sqrt n)$ памяти.
$G = \mathbb{F}_{31}^*$, $g = 3$, $y = 24$, $n = 30$, $s = 6$.
$S = \{(24,0), (10,1), (30,2), (28,3), (22,4), (4,5)\}$, $T = \{(1,0), (16,6), (8,12), (4,18), (2,24), (1,30), (16,36)\}$.
Совпадение $(4, \cdot)$: $x = 18 - 5 = 13$.

$\rho$-алгоритм Полларда для ДЛ

Разбиение $G = A \cup B \cup C$ по $z \bmod 3$. Функция: $$f(z) = \begin{cases} yz, & z \in A \\ z^2, & z \in B \\ gz, & z \in C \end{cases}$$

Инвариант $z_i = g^{\alpha_i} y^{\beta_i}$. При коллизии $z_i = z_{2i}$: $$x = \frac{\alpha_{2i} - \alpha_i}{\beta_i - \beta_{2i}} \pmod n.$$

Сложность: $O(\sqrt n)$ операций, $O(1)$ памяти (преимущество перед BSGS).

Алгоритм Полига-Хеллмана

Эффективен, когда $n = p-1 = \prod p_i^{\alpha_i}$ — $B$-гладкое.

  1. Для каждого $p_i$: таблица $r_{p_i, j} = g^{j(p-1)/p_i} \pmod p$, $j = 0, \ldots, p_i - 1$.
  2. Для каждой силы $p_i^{\alpha_i}$: найти $x \bmod p_i^{\alpha_i}$, определяя цифры $x_0, x_1, \ldots$ в $p_i$-ичной записи.
  3. Собрать $x$ по Китайской теореме об остатках.
$2^x \equiv 62 \pmod{181}$. $q = 180 = 2^2 \cdot 3^2 \cdot 5$.
Система $x \equiv 0 \pmod 4$, $x \equiv 1 \pmod 9$, $x \equiv 0 \pmod 5$ даёт $x = 100$.

Index Calculus (алгоритм Адлемана)

  1. Факторная база $\mathcal{FB} = \{q_1, \ldots, q_k\}$ — малые простые $\leq B$.
  2. Этап 1: случайные $k_j$, ищем пока $g^{k_j} \bmod p$ раскладывается по $\mathcal{FB}$. Получаем уравнения $k_j \equiv \sum a_{j,i} \log_g q_i \pmod{p-1}$.
  3. Этап 2: решив СЛАУ, находим $\log_g q_i$ для всех $q_i$.
  4. Этап 3: ищем $r$: $y g^r \bmod p$ раскладывается. Тогда $\log_g y = \sum \beta_i \log_g q_i - r$.

Сложность: $L_p[1/2, c]$. Метод COS (Коперсмит-Одлыжко-Шреппель) — оптимизированная версия.

Лекция 6. NFS для дискретного логарифма

NFS для ДЛ в $\mathbb{F}_p$ — аналог NFS для факторизации, но сложнее.

$\alpha$ — алгебраическое над $K$, если $\exists f \in K[X] \setminus \{0\}$: $f(\alpha) = 0$. $L = K(\alpha)$ — алгебраическое расширение, $[L:K]$ — степень.
Кольцо целых $\mathcal{O}_L$ — множество $\alpha \in L$, являющихся корнями нормированных многочленов из $\mathbb{Z}[X]$.
Для числового поля $\mathbb{A}$ степени $n$:
  • Любой главный идеал в $\mathcal{O}_\mathbb{A}$ имеет вид $\alpha \mathcal{O}_\mathbb{A}$.
  • Существует однозначное разложение идеала на простые идеалы.

Вводят логарифмическую функцию $\lambda(\gamma) = \gamma^k - 1 \pmod{r^2 \mathcal{O}_\mathbb{A}}$, являющуюся гомоморфизмом в аддитивную группу $r\mathcal{O}_\mathbb{A}/r^2\mathcal{O}_\mathbb{A}$. Собирают соотношения, проецируют в $\mathbb{F}_p$, решают СЛАУ как в index calculus.

Сложность: $L_p[1/3, c]$.

NFS — глубоко теоретический алгоритм. На зачёте достаточно: понимать идею (работа в кольце целых алгебраических чисел), знать сложность $L_p[1/3, c]$, понимать применимость (очень большие $p$, $> 10^{100}$).

Лекция 7. Стратегии просеивания

Главные задачи при реализации NFS:

  • Определение факторных баз (рациональной и алгебраической).
  • Набор гладких соотношений — просеивание (самая затратная часть).
  • Решение СЛАУ — миллионы уравнений, методы Block Lanczos / Block Wiedemann.

Линейное просеивание

Ищут пары $(c, d)$ такие, что идеал $(c + d\alpha)$ — гладкий, т.е. раскладывается в произведение степеней простых идеалов. Условие эквивалентно: $N(c + d\alpha)$ — $B_2$-гладкое число.

Подбор многочленов

Для NFS нужен $f(X)$ степени 3–6 с хорошими статистическими свойствами: малые коэффициенты, много вещественных корней. Методы: base-m expansion, Монтгомери-Мёрфи.

Лекция 8. Эллиптические кривые

Определение

Эллиптическая кривая над полем $\mathbb{F}$ — неособая кривая 3-го порядка с хотя бы одной точкой. В форме Вейерштрасса: $$Y^2 + a_1 XY + a_3 Y = X^3 + a_2 X^2 + a_4 X + a_6.$$ Для $\mathrm{char}(\mathbb{F}) \neq 2, 3$: сводится к $Y^2 = X^3 + aX + b$.
Неособость: дискриминант $\Delta = -16(4a^3 + 27b^2) \neq 0$.
$j$-инвариант: $j(E) = 1728 \cdot \frac{4a^3}{4a^3 + 27b^2}$. Особые случаи: $j = 0 \Leftrightarrow Y^2 = X^3 + b$; $j = 1728 \Leftrightarrow Y^2 = X^3 + aX$.

Сложение точек

$P + Q = R = (x_3, y_3)$:
  • $P \neq \pm Q$: $\lambda = \frac{y_2 - y_1}{x_2 - x_1}$, $x_3 = \lambda^2 - x_1 - x_2$, $y_3 = \lambda(x_1 - x_3) - y_1$.
  • $P = Q$ (удвоение): $\lambda = \frac{3x_1^2 + a}{2y_1}$, $x_3 = \lambda^2 - 2x_1$, $y_3 = \lambda(x_1 - x_3) - y_1$.
  • $P = -Q$: $P + Q = \mathcal{O}$.
$E(\mathbb{F}_q)$ с операцией сложения — абелева группа. Нейтральный — $\mathcal{O}$ (бесконечно удалённая точка).

Теорема Хассе

Для $E(\mathbb{F}_q)$, где $q = p^n$: $$|E(\mathbb{F}_q)| = q + 1 - t, \quad |t| \leq 2\sqrt q.$$ $t$ — след Фробениуса.

Алгоритм Шуфа

Вычисляет $|E(\mathbb{F}_q)|$ за полиномиальное время. Идея: считать $t \bmod \ell$ для малых простых $\ell$, восстановить $t$ по КТО. Использует деление многочленов и уравнение Фробениуса.

Сложность: $O(\log^8 q)$. Ключевой алгоритм для генерации ECC-кривых.

ECDLP

ECDLP: для $P, Q \in E(\mathbb{F}_q)$, где $Q = kP$, найти $k$.

Лучший классический метод — $\rho$-Полларда или BSGS: $O(\sqrt n)$. Субэкспоненциальных методов для общих EC не существует — отсюда криптографическая эффективность ECC.

Квантовая атака и постквантовая криптография

Алгоритм Шора решает и факторизацию, и DLP (включая ECDLP) за полиномиальное квантовое время. Все классические асимметричные криптосистемы — под угрозой в пост-квантовом мире.

SIDH (Supersingular Isogeny DH) — на изогениях суперсингулярных EC. Взломан в 2022 (Castryck-Decru) — не используется.

CSI-FiSh — подпись на изогениях обычных кривых. Активно изучается.

Классы сложности

$L_N[\alpha, c] = \exp\left(c (\log N)^\alpha (\log \log N)^{1-\alpha}\right)$:

  • $\alpha = 0$: полиномиальная $(\log N)^c$
  • $\alpha = 1$: экспоненциальная $N^c$
  • $0 < \alpha < 1$: субэкспоненциальная
