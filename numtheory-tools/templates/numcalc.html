🔢 Калькулятор числовых свойств

Выдаёт всё сразу про число n: разложение, φ(n) и λ(n), число и список делителей, первообразные корни (если n простое), B-гладкость.
Порядок элемента a по модулю n — наименьшее k, при котором a^k ≡ 1 (mod n). Работает, когда НОД(a, n) = 1. Перебираем делители λ(n) в порядке возрастания.
Символ Лежандра (a/p) для простого p: +1 если a — квадратичный вычет, −1 если невычет, 0 если p | a. Вычисляется по критерию Эйлера: a^((p−1)/2) mod p. Если вычет — найдём корни через Тонелли-Шенкса.
Символ Якоби (a/n) для нечётного n — обобщение символа Лежандра. Если −1, значит a — точно квадратичный невычет по mod n. Если +1 — может быть или нет.
Число называется B-гладким, если все его простые делители ≤ B. Используется в алгоритмах p−1 Полларда, Index Calculus, квадратичного решета.
