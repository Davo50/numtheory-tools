📐 НОД — алгоритмы Евклида

Классический алгоритм Евклида (Л1 стр. 8): последовательное деление с остатком a = b·q₁ + r₁, b = r₁·q₂ + r₂, … Последний ненулевой остаток = НОД.
Бинарный алгоритм Евклида (Л1 стр. 18): без деления в столбик, только |a−b| и деление на 2. ⚠️ В псевдокоде со слайда делится только a на 2, поэтому для пары чётных чисел может терять общий фактор 2. Программа предупредит, если это произойдёт.
НОД нескольких чисел: применяется классический Евклид пошагово по цепочке НОД(a,b,c,…) = НОД(НОД(a,b),c,…). Удобно для задачи «Евклид для 10 чисел».
